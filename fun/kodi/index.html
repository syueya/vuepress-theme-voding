# Kodi安装配置

# 1、Kodi 介绍与安装

# 1、介绍

Kodi 和 Infuse 类似，都是本地媒体库管理软件。不同的是，Kodi 免费！

虽然 Kodi 免费，但它的功能却非常强大，解码能力也极其优秀。可惜有点复杂，加上其反人类的操作逻辑，常常让新手一脸懵逼的放弃。

# 2、软件安装

从 Kodi官网 (opens new window) 下载 并安装，安装完成后打开kodi软件，点击继续。

然后对于一些权限点击允许，并进入kodi页面。

# 2、Kodi 使用

# 1）默认皮肤更改中文

如果需要安装地平线皮肤的话可以不进行这一步的更改，到时候更换了皮肤再更改。

点击 系统设置（齿轮图标）

进入 Interface

选择 皮肤 skin，将 Fonts 字符集 修改为 Arial based。这一步非常关键，如果没有选择Arial字体，那么调整为中文后会出现乱码。

进入 区域Regional， 在语言 Language里修改语言为 Chinese-(SimSimple) 简体中文，等待安装即可。

如果language里只有一个English，没有附带中文语言，也不用紧张，我们可以从插件库自行安装：返回到主界面，点击Add-on（插件），找到“Install From Repository”，再依次选择Look & feel→language，找到中文简体Chinese（Simple），安装后启用，即可启用中文界面。

如果不能修改为中文的话，重启一下kodi： 返回kodi主页，点击开关，然后点击跳出的Exit，退出kodi。

顺便添加一下中文键盘chinese basePY，以便Kodi输入中文。

# 2）插件安装设置

默认从未知来源安装插件是关闭的，所以我们先得打开未知来源。

点击系统system，进入系统设置。

选择插件Add-ons，将未知来源打开。

打开未知来源后，把更新插件选择为从任何库。

# 3）音频设置

回音壁和音响用户，其他用户不用看保持默认即可。

默认是基础设置，需要左下角的齿轮下点击3下后切换到专家模式。

KODI默认的声道数为2.0，如果就是用电视机喇叭，那么这里不需要调整。如果使用回音壁或者家庭影院，需要修改成匹配所用家庭影院系统的声道数。例如，如果你正在使用7.1声道的回音壁或者家庭影院，就选择7.1声道，这个选项会影响LPCM和多声道AAC音轨的表现：

# 4）播放器设置

因为电影和大部分电视都是24P帧率，在使用4K 60hz的电视观看时会因为帧率不同而产生卡顿。（120hz电视可以做5:5 pulldown所以不会卡顿）

不过大部分电视机都有电影模式，只要检测到24帧信号，就会自动激活电影模式。所以，我们要开启帧率同步，开启方式是：设置→播放器→视频，调整显示刷新率，选择“在开始/停止时”

这样，播放电影时，电视机就能自动进入电影模式；在电影结束后，又会恢复60hz，UI更顺滑。

# 3、jellyfin插件

# 1）kodi安装jf插件

点击插件

选择从zip文件安装插件。

选择nas中的repository.jellyfin.kodi.zip 文件进行安装。

等会右上角跳出件已经安装。

我们继续点从库安装：

选择jf插件。

选择视频插件。

这里的jellycon是新开项目目前bug比较多，我们用第二个。

点击安装。

第一次安装会跳出服务器添加引导，按引导添加服务器。

输入jf服务器地址。

输入账号密码。

这里选择默认即可。

安装完成右上角会出现提示，然后点击要添加的影视库。等待服务器同步后就能看见添加的影视了。

# 2）jf安装kodi插件

然后我们回到jellyfin后台安装kodi同步插件，安装完成后得重启jf，状态才是active。

点击插件下竖着的三点进入菜单，选择设置。

选择要同步的媒体点击保存。

# 4、地平线皮肤安装

# 1、安装插件库

两种方法任选一样即可。

1）zip文件安装

点击Add-ons (插件)

选择Install from zip file (从zip文件安装)。PS：Kodi只能安装zip格式插件，如果遇到其他格式如RAR等需要把里面的zip解压出来。

因为我们的插件放在nas里，所以选择SMB来连接nas。

选择自己的机器

这里是输入绿联网络服务里的账号密码。

然后进入nas文件夹，找到之前存放插件库的目录。

2）官方源安装

在系统设置，进入文件管理。

点击添加源。

点击“无”，然后手动输入地址https://jurialmunkey.github.io/repository.jurialmunkey，自动命名为repository.jurialmunkey后点确定。

成功完成后文件管理会增加一个“repository.jurialmunkey”的文件地址。

回到系统设置，进入插件。

点击从zip文件安装。

进入我们刚刚添加的repository.jurialmunkey。

选择zip文件点击ok。

# 2、安装插件

PS:安装提前设置好梯子。

右上角显示插件库安装完后，点击从库安装。

选择jurialmunkey。

选择 界面外观Look and feel。

选择皮肤skin。

选择地平线2皮肤（Arctic Horizon2）。

点击安装。

跳出所需的依赖，点击ok。

如果一直安装不上的话，先去插件里把已有的这些依赖全部卸载掉，再来重新安装。

PS：对于一些依赖比如Tmdbmoviehelper安装需要有梯子。另外它的使用也需要，否则会报错，如果不想显示报错可以在插件列表中选中该插件按菜单键禁用，不要进该插件设置里禁用。

皮肤下载中。

# 5、地平线皮肤使用kodi

# 1、切换皮肤

皮肤下载完成后询问是否切换到此皮肤，点击yes。

切换后询问是否保持使用此皮肤 点击yes。

皮肤启动中。

经过一小会加载后，皮肤安装完成了，这就是安装完毕的初始样式。

# 2、切换中文

点击右上角的设置图标，选择settings设置。

选择皮肤skin，把字符集fonts改成CJK-Chinese/Japanese/Korean。

选择区域Regional，把语言language由默认的英语改成Chinese（Simple).

更改完成，界面变成了中文。

# 3、安装依赖

然后选到第一项把缺失依赖全部安装

# 4、添加影视库

推荐通过jf插件来导入视频。

点击视频-文件。

点击添加视频。

点击浏览。

这里有多个方式可以选择自己的影片保存的位置，点击添加网络位置。

然后输入绿联的IP地址和绿联网络服务里的账号密码。

再到这里找到刚刚设置的IP地址，点进去找自己的媒体库。Ps：Kodi什么协议都可，但是SMB出卡顿的概率会更大，FTP相对卡顿概率比较小。

选择好媒体文件后点击确定。

进行一些内容的设置：

  • 该目录包含选择电影或者电视。
  • 选择信息提供者：Local information only（只读取本地信息，如果有刮削好的话建议这个）或者TMDB。
  • 在下方勾选电影在以片名命名的单独目录中和嵌套扫描。

设置完成后点击确定。

点击是来刷新信息。

右上角会开始扫描内容。

等一会，右上角显示扫描结束。

点击刚刚增加的华语电影进去，等一会可以看到海报墙。

添加完影视库后可以点击右上方的设置-视图。

可以选择不同的视图设置。

影视库的视频海报墙设置好了。

# 8、字幕插件

把repository.xbmc-addons-chinese.zip 文件放到nas中，选择从zip文件安装插件，等会右上角跳出件已经安装。

我们继续点从库安装：

选择刚刚已经安装的这个字幕插件：

点击字幕选项

三个都安装完成

# 10、遥控器插件

kodi默认遥控器配置比较反人类建议修改一下。

点击搜索。

搜索Keymap Editor插件。

点击安装

安装后右上角有提示。

点击运行。

点击编辑

选择全屏视频

选择Playback

选择停止

编辑键（这时会跳出来一个倒计时，在5秒内你需要按下遥控器对应的按键（这里就是返回键））
